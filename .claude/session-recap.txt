7. ARCHITECTURE RULES (from CLAUDE.md, must follow)
-------------------------------------------------------------------------------

Monorepo:
  packages/mcp-server/   npm package @unclick/mcp-server
  src/                   React website (Vite + TS)
  api/                   Vercel serverless functions

4 meta-tools pattern (NEVER register per-tool MCP endpoints):
  unclick_search      find tools by keyword
  unclick_browse      list all tools, optionally by category
  unclick_tool_info   endpoints and params for a tool
  unclick_call        execute any endpoint with parameters

Adding a new tool:
  1. api/*-tool.ts       Vercel handler + endpoint logic
  2. tool-wiring.ts      Add name, description, category, endpoint mapping
  3. Tools.tsx           Add tile

Style:
  - No em dashes anywhere
  - No per-tool MCP registrations
